### Introduce
네이버를 기반으로 자동출입명부 (QR체크인)을 자동으로 생성합니다.<br/>
전자출입명부에 특화된 앱으로 보다 편리하게 사용하실 수 있습니다.
* **`웨어러블(Wear OS)`을 지원합니다.**<br/>
  휴대폰이 없어도, 인터넷이 되는 웨어러블이라면 자동출입명부를 웨어러블로 보다 쉽게 하실 수 있습니다.
